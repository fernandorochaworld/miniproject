# Currencies API

![Express Logo](https://upload.wikimedia.org/wikipedia/commons/6/64/Expressjs.png)

## Description
API express for maintain currencies in the Coding in Colour project.


## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start
```


## Start Docker Compose

```bash
$ docker-compose up -d
```
